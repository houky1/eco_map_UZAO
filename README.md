## Цель проекта
Проект посвящен разработке собственного устройства на основе arduino-nano для оценки качества воздуха, последующего сбора проб с точек локации, обработке данных и визуализации результатов
с помощью создания экологической карты Москвы.
### Результаты проекта
- Разработано устройство на основе arduino-nano для оценки качества воздуха.
- Создана система сбора данных с точек локации.
- Разработан алгоритм обработки данных.
- Создана визуализация результатов в виде экологической карты Москвы.
### Выводы
Качество воздуха в ЮЗАО г. Москвы в целом соответствует нормам, установленным в СанПиН, однако в некоторых районах наблюдаются превышения допустимых концентраций загрязняющих веществ.
